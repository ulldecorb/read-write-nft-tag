## General Info
This project basically moves between read and write scope. 
## Technologies 
+ React Native 0.72.2
+ Typescript 4.8.4
+ React-native-nfc-manager 3.14.5
## Set up

To check device connected:
```
$ adb devices
```
Once you make sure device is connected, launch the app with:
```
$ npm run android
```
